# grade_12

Google Colab notebook for Ethiopian Grade 12 entrance-exam pattern analysis:

- `ethiopian_exam_pattern_colab.ipynb`: End-to-end workflow to:
  - download exam PDFs,
  - run OCR extraction,
  - clean and parse question text,
  - produce per-year topic trend charts,
  - score likely next-exam topics.
- `download_ethiopian_exams.py`: Original local downloader script.
- `extract_exam_text_ocr.py`: Original local OCR extraction script.
